# gauge-sample
Gaugeのサンプルプログラムです

# 概要
JJUG 2017 FALLで使用するものです  

## ライブラリ
使用ライブラリは以下のとおりです

|ライブラリ|URL|
|-----------|------------|
|Gauge(Gauge-Java)|<https://getgauge.io/index.html>|
|Selenide|<http://selenide.org/>|
|JUnit4|<http://junit.org/junit4/>|

## ドライバ
Chrome driverを利用しています。

## 起動方法
プロジェクトにディレクトリを移動して以下コマンドを叩いてください
```
mvn test
```
